# USB CONFIGURATION

Requirements :

1) 32 GB
3) interface type: 3.0

Follow the steps :

> Execute the "disks" program

![img1](img/usb/1.png)

> Choose the usb drive to configure and press format disk

![img2](img/usb/2.png)

> Choose "Compatible with modern systems and hard disks > 2TB(GPT)"

![img3](img/usb/3.png)

![img4](img/usb/4.png)

> Create new partition (size == all memory)

![img5](img/usb/5.png)

> Volume name: any
> Erase: none
> Type: other

![img6](img/usb/6.png)

> Choose F2FS
![img7](img/usb/7.png)

> Format created partition

![img8](img/usb/8.png)

> The new name must be 01-home

![img9](img/usb/9.png)
