# Server requirements

Features required:

- Dedicated hardware
- Linux compatible

| Component | Minimum specifications                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------------------ |
| Processor | 8 threads x86-64 (64 bits), [2200 single thread performance](https://www.cpubenchmark.net/singleThread.html) |
| Memory    | 16 GB DDR4                                                                                                   |
| Disk      | 2 \* 500 GB SSD                                                                                              |
