# Server requirements

Features required :
- Linux compatibility
- UEFI
- Secure boot
- TPM 2.0

| Component | Recommended specifications  |
| --------- | --------------------------- |
| Processor | 12 threads x86-64 (64 bits) |
| Memory    | 16 GB DDR4                  |
| Disk      | 2 * 500 GB SSD              |
