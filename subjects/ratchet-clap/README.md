## Ratchet👏Clap👏

Let's do the opposite !

### Instructions

Use the provided `words` variable of an array of strings and **join** it using
the 👏 emoji.

Also add one more 👏 at the end of the sentence, for good measure !

Assign the result in a `ratchetClap` variable.
