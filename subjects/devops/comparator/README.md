## comparator

### Instructions

Create a script `comparator.sh` which will verify if the first given argument is bigger, smaller or equal than the second given argument.
You must print te output like the example in usage.

### Usage

```console
$ ./comparator.sh 6 14
6 < 14
$ ./comparator.sh 29 3
29 > 3
$ ./comparator.sh 12 12
12 = 12
$
```
