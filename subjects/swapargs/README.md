## swap-args

### Instructions
Write a program that takes two arguments and swaps them 
- If the arguments are not 2 print (`'\n'`).
- Separate the arguments with (`' '`)
- Print (`'\n'`) at the end of the output.

### Example

```console
$ go run . | cat -e
$
$ go run . 1 2 | cat -e
2 1$
$ go run . "World" "Hello" | cat -e
Hello! World$
```
