## Unicode Technical Report 35

### Subject

Implement a `format` function that takes 2 arguments:

- a valid javascript Date object
- a format string that handle all of the following fields:
  - y
  - yyyy
  - G
  - GGGG
  - M
  - MM
  - MMM
  - MMMM
  - d
  - dd
  - E
  - EEEE
  - h
  - hh
  - m
  - mm
  - s
  - ss
  - H
  - HH
  - a
