## canyoucount

### Instructions

Your program will receive some arguments. Count how many characters they have in total and print them.

- If the number of arguments is invalid it should print `0`.

### Usage

```console
$ go run . "hello" "how are you?" | cat -e
17$
$ go run . "hi" | cat -e
2$
$ go run . | cat -e
0$
```
