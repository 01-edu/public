## Plain Sum

### Instructions

Create a function named `plainSum`, which accepts 3 `int` parameters, and returns their sum.

Functions in Dart are declared like this:

```dart
bool isEven(int num) {
  return num % 2;
}
```

### Usage

Here is a possible program to test your function :

```dart
void main() {
  var result = plainSum(1, 2, 3);
  print(result);
}
```

And its output :

```console
$ dart test.dart
6
$
```
